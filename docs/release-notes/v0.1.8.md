# WenZi v0.1.8 Release Notes

## What's New

- **Plugin system**: management UI, registry, install/update/uninstall with version pinning
- **cc-sessions plugin**: Claude Code session browser with preview, subagent linking, identicon avatars, disk cache
- **Settings panel**: migrated to WKWebView, STT/LLM provider forms with verify & save
- **i18n**: full Chinese + English localization with language switching
- **Launcher**: System Settings search source, localized source names
- **Correction-driven vocabulary**: CorrectionTracker with ASR hotword and LLM vocab injection
- **Security**: provider API keys stored in macOS Keychain
- **Bug fixes**: cancel/restart race condition, test suite optimized (76s → 47s), multiple code review fixes (#58–#64, #67)

# 闻字 v0.1.8 版本说明

## 新功能

- **插件系统**：插件管理 UI、注册表、安装/更新/卸载、版本固定
- **cc-sessions 插件**：Claude Code 会话浏览器，支持预览、子代理链接、identicon 头像、磁盘缓存
- **设置面板**：迁移到 WKWebView，STT/LLM 提供商表单支持验证并保存
- **国际化**：完整中英文本地化，支持语言切换
- **启动器**：系统设置搜索源、搜索源名称本地化
- **纠错驱动词库**：CorrectionTracker，自动注入 ASR 热词和 LLM 词库
- **安全性**：提供商 API 密钥存储到 macOS 钥匙串
- **Bug 修复**：取消/重启竞态条件、测试套件优化（76s → 47s）、多项代码审查修复（#58–#64、#67）
